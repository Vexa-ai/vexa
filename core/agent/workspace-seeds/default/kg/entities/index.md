# entities

The knowledge graph's typed entities, one directory per type (`<type>/<slug>.md`). Each type has its
own `index.md` listing its entities. The examples below show the SHAPE — required frontmatter
(`type`, `id`, `title`), `[[wikilinks]]` between entities by title, and one directory per type. Replace
them with your own knowledge (or delete them) as you go.

- [person](person/index.md) — [[Jane Liu]]
- [company](company/index.md) — [[Acme Corp]]
- [meeting](meeting/index.md) — [[Acme intro call]]
