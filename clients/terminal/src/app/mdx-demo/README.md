# mdx-demo (standalone renderer showcase)

`/mdx-demo` — no-gateway demo page for the `ui-kit/MdxDoc` workspace renderer:
raw agent-authored markdown (left) next to the live rendered interface (right),
including the closed component registry (`Note`, `Card(Group)`, `Steps`,
`Tabs`, `Wikilink`), workspace-internal link handling, and the plain-markdown
fallback for malformed MDX. Useful for eyeballing renderer changes without a
running backend.
