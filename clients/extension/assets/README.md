# extension/assets — static extension assets

Copied verbatim into `dist/` by `build.mjs`. Referenced by the manifest and the
side-panel HTML.

- `vexalight.svg` / `vexadark.svg` — the Vexa wordmark, light- and dark-theme variants.
- `icons/` — the toolbar / extension icons (16, 32, 48, 128 px PNGs).
