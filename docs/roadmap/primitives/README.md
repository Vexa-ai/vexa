# primitives — the irreducible building blocks (Mintlify pages)

The seven primitives the whole runtime composes from: `workspace` (the user git repo), `agent` (the
governed turn), `runtime` (spawn/execute), `scheduler` (cron/one-shot), `integration` (event in / action
out), `stream` (live output), `identity` (who acts). `overview.mdx` ties them to the `unit.v1` envelope.
