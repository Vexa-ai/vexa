---
title: "Agent"
description: "A generic CLI agent selected by a runner. Claude-code is one. Untrusted by design."
---

The **Agent** does the work: given mounted [workspaces](/primitives/workspace), a toolbelt, and a starting
point, it acts and commits. It is selected by a **runner** — the swappable adapter that knows how to drive
a specific CLI agent.

- **Runner** = `{ image, argv template, stream format, session path }`. **claude-code** is the first
  runner; aider or others plug in; BYO-inference is a runner setting. (We renamed this from "profile".)
- **Starts from an entry point or a session** — a fresh task (entry point: a workspace path or inline) or
  a [resumed session](/architecture/execution) (a file in the workspace).
- **Untrusted.** The agent is outside the trust boundary — prompt-injectable, granted **zero** enforcement
  power. It carries a signed token; the [boundaries verify it](/architecture/identity-and-trust).
