title: "The seven primitives"
description: "The irreducible building blocks — and how they compose into a dispatch."

Seven primitives. Everything in the product is one of them, or a composition of them.

| # | Primitive | One line |
|---|---|---|
| 1 | [Workspace](/primitives/workspace) | a git **folder in a bucket**, with an `id` + access **mode** (ro/rw); encrypted at rest |
| 2 | [Agent](/primitives/agent) | a generic CLI agent (a **runner** — claude-code is one); untrusted |
| 3 | [Runtime](/primitives/runtime) | **isolated, sandboxed** containers; docker→k8s; **TTL-on-idle** |
| 4 | [Scheduler](/primitives/scheduler) | **redis**; dispatches on a schedule entry / integration event / now |
| 5 | [Integration](/primitives/integration) | persistent; **inbound** (watch→fire) + **outbound** (MCP tool); cred-gated |
| 6 | [Stream](/primitives/stream) | a per-dispatch **redis Stream** → websocket; durable + replayable |
| 7 | [Identity](/primitives/identity) | the **trust spine**: authenticate · mint signed token · attest workload · verify at boundaries |

## How they compose — the dispatch

```
dispatch = {
  identity:   { subject, launcher, signed-token },   // on-behalf-of · who-triggered · the proof   (#7)
  runner:     "claude-code",                          // which agent CLI + how to drive it          (#2)
  workspaces: [ {id, mode} ],                          // mode = the write-access source of truth     (#1)
  tools:      [ name ],                                // each auto|gate                              (#5)
  start:      entrypoint(path | inline)  |  session(ref),
}
   → Scheduler(#4) fires it → Runtime(#3) spawns an isolated container, mounts the workspaces(#1)
     + brokers creds(#5,#7) → the Agent(#2) runs, streams(#6); idle ⇒ reaped.
// derived, not stored:  trust = (all workspaces ro && irreversible tools gated);  output = unit:<id>:out
```

This is the **`unit.v1`** contract — the one envelope every trigger emits. A chat *now*-dispatch:

```json
{
  "identity":   { "subject": "u_jane", "launcher": "user:u_jane" },
  "runner":     "claude-code",
  "workspaces": [ { "id": "u_jane", "mode": "rw" } ],
  "trigger":    "message",
  "context":    { "kind": "none" },
  "start":      { "entrypoint": { "inline": "summarize my open tasks" } }
}
```

The Runtime injects the minted `identity.token` at spawn; `output` (`unit:<id>:out`) and `trust` are
**derived, never stored**. Contract: `core/agent/contracts/unit.v1/`.

## Not primitives (they are *made of* these)

- **Plan** = an entry point — a file path or inline task (a [Workspace](/primitives/workspace) concern).
- **Trigger / event** = an input to the [Scheduler](/primitives/scheduler); an event = an
  [Integration](/primitives/integration) firing it.
- **Routine** = an entry in the [Scheduler](/primitives/scheduler).
- **Task / note / entity** = files in a [Workspace](/primitives/workspace).
- **Proposed action / card** = a frame on the [Stream](/primitives/stream); the spine of
  *propose → approve → trusted apply* (see [Governance](/architecture/governance)).
- **Tool** = the outbound face of an [Integration](/primitives/integration).
