---
title: "AI routines"
description: "Agents that run unattended — on a schedule, or when something happens."
---

A routine is an entry in the [scheduler](/concepts#scheduler): a trigger plus a plan. The agent runs
unattended and commits to the [workspace](/concepts#workspace) — no one watching, output replayable later.

- **On a schedule** — "every weekday 8am, brief me from overnight activity."
- **On an event** — "when an email arrives, triage it into tasks" (propose-only, since email is
  [untrusted input](/core/agents)).
- **After a meeting** — "summarize and file the follow-ups."

Same agent, same isolation — only the trigger changes.
