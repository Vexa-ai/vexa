title: "Org knowledge"
description: "A scheduled dispatch that merges person folders into an org folder, in git."
---

Org knowledge is triaged **in git, like a software project**. A scheduled [dispatch](/architecture/dispatch)
diffs each person's new files against an **org folder** ([workspace](/primitives/workspace) with `company`
ro + `org` rw) and opens a reviewable merge proposal (sign/reject). The "git like a software project"
thesis is load-bearing here — identity resolution + conflict policy on entities.
