---
title: "Knowledge chat"
description: "Chat with an agent that has your whole workspace as context."
---

Chat is a [dispatch](/core/agents) fired *now*: you send a message, an agent works your
[workspace](/concepts#workspace) — every meeting, email, and note already compiled there — and answers,
streaming back live. Because input from you is trusted, the same agent can also write: record a decision,
update an entity, draft a doc.

Not retrieval over a document store — the workspace is a maintained Markdown knowledge base the agent
reads and edits like a developer in a repo.
