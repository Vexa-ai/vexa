---
title: "Knowledge"
description: "Tasks, notes, entities — just files in the workspace."
---

There is no "knowledge service". Tasks, notes, people, companies, meeting notes are **files** the agent
writes in the [workspace](/primitives/workspace) git folder. The terminal renders *views* (a Tasks list, a
graph) over those files. Structure is **emergent and freeform** — never dictated by code or a contract.
